# Cross-posting (Thingiverse / Printables)

## Required before publishing
- At least one real photo exists in `images/` and is referenced by README.
- `exports/` contains the exact artifacts youâ€™re uploading.
- Artifact filenames are stable and descriptive.
- License is present.
- Thingiverse/Printables listing license is set to **MIT** (matches [LICENSE](../LICENSE)).

## Suggested upload package
Create a ZIP containing:
- `exports/stl/*.stl`
- `README.md`
- `images/printed_plate_1.jpg`

## Listing text
Keep it short and technical:
- what the plate is for
- the hole size range and step
- what the labels mean
- recommended print settings

## Versioning
- Tag GitHub releases (e.g. `v1.0.0`).
- Cross-post updates SHOULD match the GitHub release notes.
